# GaebalSaebal_Project

## 💁 프로젝트 소개

개발새발은 예비 개발자, 현직 개발자가 함께 이용할 수 있는 사이트입니다. 이용자는 퀴즈 게임을 통해 개발 지식을 학습할 수 있고, 퀴즈 게임을 통해 획득한 포인트로 가상의 상품을 구매할 수 있습니다. 그리고 커뮤니티에 글을 작성해 각종 정보와 일상을 함께 나눌 수 있습니다.
